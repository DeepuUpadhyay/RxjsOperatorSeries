Migration Guide

Pluck to Map Migration:
Old (Deprecated)
// Simple property
source.pipe(pluck(''name''))

// Nested property  
source.pipe(pluck(''user'', ''profile'', ''email''))
New (Recommended)
// Simple property
source.pipe(map(obj => obj.name))

// Nested property
source.pipe(map(obj => obj.user?.profile?.email))

Important Notes

Best Practices:
  • Migration: Replace pluck with map for better type safety and future compatibility
  • Null Safety: Use optional chaining (?.) with map to handle undefined properties
  • Performance: Map provides better performance than pluck in modern RxJS
  • Type Safety: Map with TypeScript provides better type inference than pluck
